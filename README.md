# alex
alexはISOビルダーの一種であり、Ubuntu系ディストリビューションを作成するためツールです。

カスタマイズ方法はCustom-alex.mdを参照してください。

# ビルド方法
```
sudo apt install -y git
git clone https://github.com/Plasma-Linux/alex/
cd alex
sudo chmod 775 ./build.sh
sudo ./build.sh
```
ビルドが完了しているとout以下にISOファイルがあるはずです。
全自動ではないことに気を付けてください。
